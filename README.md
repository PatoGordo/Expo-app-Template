# Template App RN com Expo

---

## Comandos

### Inicia servidor de desenvolvimento (ExpoGo)

```
yarn dev
```

### Inicia app no emulador ou cabo no android

```
yarn android
```

### Inicia app no emulador ou cabo no ios

```
yarn ios
```

### Inicia um build no cloud do expo (EAS)

```
yarn eas:build
```
